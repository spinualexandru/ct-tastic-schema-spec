# ct-tastic-schema-spec
A json schema for allowing to have typings on your Tastic Schemas


## Usage
```json
{
  "$schema": "https://raw.githubusercontent.com/spinualexandru/ct-tastic-schema-spec/refs/heads/main/tastic.schema.json",
  "type": "object",
  "properties": {
    "foo": {
      "type": "string"
    }
  }
}
```
